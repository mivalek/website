Standard HTTP response codes were defined by the OG web developer Tim Berners-Lee in 1992, based on the ones used for File Transfer Protocol. All 4xx responses code for client errors, such as unauthorised request (401) or request timeout (408). The 404 error occurs when the server cannot find the page requested by the client. This can be due to the page being moved or a typo in the URL.
